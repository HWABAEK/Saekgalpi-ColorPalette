# 💡 About
한글로 구성된 **색갈피(Saekgalpi-ColorPalette)** 에서 손쉽게 색을 찾아서 사용해 보세요!
```
1. 배색과 피커가 같이 있는 색갈피
2. 사용자가 커스텀 가능한 색갈피
3. 공유가 가능한 색갈피
    - 공유 방법
        1. 헥사코드
        2. RGB
        3. 이미지 저장
        4. url 카피
```
